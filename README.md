## üìå How To Install?

- Clone This Repo
```
git clone https://github.com/eexxprd/frontend-thekingfurniture
```
- Go To Folder Repo
```
cd frontend-thekingfurniture
```
- Install Module
```
npm install
```
- Type ``` npm run dev ``` To Start The Project
- Set file .env ```http://localhost:5000``` from backend
